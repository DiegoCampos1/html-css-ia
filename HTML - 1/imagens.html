Imagens

É preciso tomar cuidado com os direitos autorais das imagens, não utilizar imagens sem permissão

Cuidado com o tamanho das imagens, utilizar imagens com resoluções adequadas, não utilizar imagens com resoluções muito grandes pois ele pode demorar muito para carregar

JPEG é um formato de compactação de imagens, ele permite reduzir o tamanho das imagens sem perder muita qualidade

PNG é um formato de imagem que suporta transparência, ele é mais pesado que o JPEG, mas é mais adequado para imagens com fundo transparente

GIF é um formato de imagem que suporta animações, ele é mais pesado que o JPEG e o PNG, mas é mais adequado para imagens com animações

SVG é um formato de imagem vetorial, ele é mais leve que o JPEG, PNG e GIF, mas é mais adequado para imagens com formas geométricas

WebP é um formato de imagem desenvolvido pelo Google, ele é mais leve que o JPEG, PNG e GIF, mas é mais adequado para imagens com animações

Como mudar o favicon do site

O favicon é o ícone que aparece na aba do navegador, ele é importante para a identidade visual do site

Para mudar o favicon do site, basta adicionar a seguinte linha dentro da tag head:

<link rel="icon" href="favicon.ico" type="image/x-icon">

O favicon deve estar no formato .ico, .png ou .jpg

O favicon deve ter o tamanho de 16x16 pixels ou 32x32 pixels

O favicon deve estar na raiz do site ou em uma pasta chamada "images" ou "icons"
