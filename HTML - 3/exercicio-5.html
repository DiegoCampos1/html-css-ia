Exercício 3 - Formulário de Inscrição

Monte um formulário completo com:

  • Campo para nome completo (obrigatório)
  • Email com validação (obrigatório)
  • Senha (obrigatório)
  • Confirmação de senha (obrigatório)
  • Telefone
  • Data de nascimento
  • URL do portfólio (opcional)
  • Upload de currículo (arquivo PDF)
  • Caixa de seleção para aceitar os termos
  • Botões de envio e limpar

Valide os campos mais importantes usando os atributos required, type e accept no input file.
