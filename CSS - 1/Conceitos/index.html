O que é CSS e por que é usado?

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar documentos HTML ou XML. Com ele, controlamos o layout, cores, fontes e espaçamentos de páginas web.

Importância: O CSS é essencial para criar sites mais bonitos, modernos e organizados. Ele separa o visual do conteúdo, facilitando a manutenção do código e permitindo que o design seja alterado sem mexer no HTML.

Separando Estilo do Conteúdo

Um dos princípios mais importantes do desenvolvimento web é separar o conteúdo (HTML) do estilo (CSS).

  • HTML: define a estrutura e o significado do conteúdo (títulos, parágrafos, listas, imagens).
  • CSS: define como esses elementos devem parecer (cores, fontes, posicionamento).

Essa separação melhora a clareza, reutilização de estilos e permite que designers e desenvolvedores trabalhem juntos com eficiência.

Estrutura de uma Regra CSS

Uma regra CSS é formada por três partes:

  • Seletor: Indica qual elemento HTML será estilizado.
  • Propriedade: Especifica o que será modificado (ex: cor, fonte, tamanho).
  • Valor: Define o valor da propriedade (ex: blue, 16px, bold).

Exemplo:


h1 {
  color: blue;
  font-size: 32px;
}

No exemplo acima, o seletor é h1, a propriedade é color e o valor é blue.
