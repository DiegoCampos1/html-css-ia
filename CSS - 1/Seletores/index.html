Seletores em CSS

Seletores são usados para escolher quais elementos HTML serão estilizados.

1. Seletor de Elemento (Tag)

Este parágrafo é estilizado com o seletor de elemento: p

2. Seletor de Classe

Esta div usa a classe .destaque

3. Seletor de ID

Este parágrafo usa o seletor de ID #importante

4. Seletor Universal

Tudo na página (corpo, listas, divs, etc.) recebe uma margem básica pelo seletor *

5. Seletor de Agrupamento

Esse h3 e o próximo h4 usam o mesmo estilo definido com h3, h4 {}

Outro título agrupado

6. Seletor Descendente (espaço)

Esse parágrafo está dentro de uma div.container e é estilizado com .container p

7. Seletor de Filho Direto (>)

Esse parágrafo é filho direto e é afetado por .pai > p

Esse parágrafo NÃO é afetado, pois está dentro de outro elemento
