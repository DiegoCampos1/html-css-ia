Modelo de Caixa no CSS

O Modelo de Caixa é um conceito fundamental no CSS que descreve como os elementos HTML são representados como caixas retangulares. Cada elemento é tratado como uma caixa que envolve:

  • Conteúdo (Content)
  • Preenchimento (Padding)
  • Borda (Border)
  • Margem (Margin)

Conteúdo (Content)

É a área onde o texto, imagem ou outro conteúdo é exibido. Suas dimensões podem ser controladas com width e height.


div {
  width: 200px;
  height: 100px;
}
Exemplo de caixa com largura de 200px e altura de 100px.

Preenchimento (Padding)

O preenchimento é o espaço interno entre o conteúdo e a borda do elemento.


div {
  padding: 10px;
}

Dica: Você pode controlar cada lado separadamente: padding-top, padding-right, padding-bottom e padding-left.

Borda (Border)

A borda envolve o conteúdo e o padding. Você pode definir a largura, estilo e cor da borda.


div {
  border: 2px solid black;
}

Principais propriedades:

  • border-width (largura da borda)
  • border-style (tipo: sólido, tracejado, etc.)
  • border-color (cor da borda)

Margem (Margin)

A margem é o espaço externo entre a borda do elemento e os elementos ao redor.


div {
  margin: 15px;
}

Dica: Assim como o padding, você pode usar: margin-top, margin-right, margin-bottom e margin-left.
