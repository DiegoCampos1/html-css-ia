Exercícios: Aplicando CSS na prática

Exercício 1 – Estilo básico com seletor de elemento

Objetivo: Criar um arquivo HTML e estilizar títulos e parágrafos usando CSS.

Instruções:

  1. Crie um novo arquivo chamado exercicio1.html.
  2. Adicione pelo menos:
    • 1 título com a tag <h1>
    • 2 subtítulos com <h2>
    • 3 parágrafos com <p>
  3. Use CSS (interno ou externo) para:
    • Mudar a cor dos títulos
    • Alterar a fonte e o tamanho dos parágrafos
    • Adicionar espaçamento entre os elementos

Exercício 2 – Estilizando um perfil fictício

Objetivo: Criar uma página de perfil fictício com seções bem definidas e aplicar estilos usando diferentes seletores (classe, ID, agrupamento, descendente).

Instruções:

  1. Crie um novo arquivo chamado exercicio2.html com um arquivo de estilos estilo.css.
  2. No HTML, crie a seguinte estrutura:
    • Um h1 com seu nome
    • Uma div com ID sobre contendo:
      • Um h2 "Sobre mim"
      • Um p com uma descrição pessoal
    • Uma section com classe habilidades contendo:
      • Um h2 "Habilidades"
      • Uma lista com pelo menos 3 habilidades
    • Uma div com links para redes sociais (LinkedIn, GitHub, etc)
  3. No CSS:
    • Use o seletor de ID para estilizar a seção #sobre
    • Use o seletor de classe para estilizar a section.habilidades
    • Use o seletor descendente (ex: .habilidades li) para formatar os itens da lista
    • Use o seletor de atributo para estilizar links externos
    • Adicione cores, espaçamento e fontes diferentes em cada parte

Dica: Capriche no visual, use a criatividade e os conhecimentos que você já tem de HTML e CSS!
