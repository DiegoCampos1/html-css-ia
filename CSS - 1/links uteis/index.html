Links úteis para aprender e praticar CSS

  • MDN Web Docs – CSS Documentação completa e confiável sobre CSS, com exemplos e explicações detalhadas.
  • W3Schools – CSS Tutorial Material introdutório interativo para praticar direto no navegador.
  • CSS-Tricks Blog com tutoriais, dicas e técnicas modernas de CSS e front-end em geral.
  • Flexbox Froggy Jogo interativo para treinar alinhamento com Flexbox.
  • CSS Battle Desafios de código para recriar layouts com o menor CSS possível.
  • CodePen Ambiente online para testar e compartilhar códigos HTML/CSS/JS com visualização instantânea.
  • Frontend Mentor Desafios reais de layout para praticar HTML e CSS com feedback da comunidade.
  • CSS Reference Referência visual com exemplos de cada propriedade CSS.
