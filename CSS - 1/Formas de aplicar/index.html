Formas de Aplicar CSS

O CSS pode ser aplicado de três formas principais: inline, interno e externo. Cada abordagem tem sua utilidade, vantagens e desvantagens.

1. CSS Inline

O estilo é aplicado diretamente no elemento HTML usando o atributo style.

Este parágrafo usa CSS inline para definir cor de texto, fundo e espaçamento.

2. CSS Interno

O CSS é escrito dentro da própria página HTML, dentro da tag <style>, localizada no <head>.

Este bloco foi estilizado com CSS interno usando uma classe chamada .interno.

3. CSS Externo

O CSS externo é armazenado em um arquivo separado (ex: estilo.css) e vinculado à página HTML usando a tag <link>.

Exemplo da linha que liga o CSS externo ao HTML (inserida no <head>): 

<link rel="stylesheet" href="estilo.css">

Vantagem: permite aplicar o mesmo estilo a várias páginas e facilita a manutenção do projeto.

Resumo:

  • Inline: Rápido e direto, mas difícil de manter em projetos grandes.
  • Interno: Útil para projetos pequenos ou testes.
  • Externo: Melhor prática para projetos reais e escaláveis.
