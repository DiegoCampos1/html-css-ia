Posicionamento de Elementos com CSS

O posicionamento em CSS define como os elementos são localizados na página e como se comportam no layout.

Os principais valores da propriedade position são: static, relative, absolute, fixed, sticky.

position: static (padrão)

O elemento é posicionado conforme o fluxo normal do HTML.

Posição: static

position: relative

Permite mover o elemento em relação à sua posição original.

Posição: relative
top: 20px; left: 30px

position: absolute

Remove o elemento do fluxo e posiciona em relação ao elemento pai mais próximo com position: relative.

Posição: absolute

position: fixed

Fixa o elemento na tela, mesmo ao rolar a página.

Posição: fixed

position: sticky

Fixa o elemento no topo durante o scroll, quando alcança a posição especificada.

Posição: sticky (top: 0)

Controle de Camadas com z-index

O z-index define qual elemento aparece por cima de outro. Só funciona em elementos posicionados (não static).

z-index: 1
z-index: 2

Resumo:
- Elementos com z-index maior ficam acima.
- Funciona apenas com relative, absolute, fixed ou sticky.
- Pode aceitar valores positivos ou negativos.
