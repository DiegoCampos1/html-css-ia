Display e Visibility no CSS

As propriedades display e visibility são usadas para controlar como os elementos HTML aparecem visualmente e como afetam o layout da página.

Propriedade display

Controla o modo como um elemento é exibido no layout:

display: block

O elemento ocupa toda a largura disponível e força quebras de linha antes e depois.

Exemplo de bloco (div)

display: inline

O elemento ocupa apenas o espaço necessário e não quebra linha.

Item 1Item 2Item 3

display: inline-block

Comporta-se como inline, mas permite definir width e height.

Caixa 1Caixa 2

display: none

Remove o elemento completamente da página (não ocupa espaço).

Você não verá isso (display: none)
display: none;

Propriedade visibility

Controla a visibilidade do elemento, mas mantém sua posição no layout.

visibility: visible (padrão)

Este elemento está visível

visibility: hidden

Este elemento está invisível, mas ainda ocupa espaço.
visibility: hidden;

Resumo:

  • display: none remove o elemento do layout completamente.
  • visibility: hidden apenas o esconde visualmente, mas o espaço ainda está lá.
