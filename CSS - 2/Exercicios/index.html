Exercício: Trabalhando com display e visibility

Objetivo: Criar uma pequena interface com elementos visuais utilizando diferentes valores das propriedades display e visibility.

Instruções:

  1. Crie um arquivo HTML chamado exercicio-display.html e um arquivo CSS separado chamado style.css.
  2. No HTML, crie os seguintes elementos dentro de <body>:
    • Um título principal <h1>
    • Três <div> com a classe caixa, com textos como "Bloco 1", "Bloco 2", "Bloco 3"
    • Dois <span> com a classe tag-inline
    • Uma <div> com a classe ocultar
  3. No CSS, faça o seguinte:
    • Use display: block para as .caixa (valores fixos de largura e altura, borda, cor de fundo)
    • Use display: inline para os elementos com .tag-inline (cor de fundo e espaçamento lateral)
    • Defina visibility: hidden para a .ocultar, mas deixe visível o espaço que ela ocupa
    • Crie uma classe chamada .sumir com display: none e aplique ela a um dos blocos
  4. Teste o comportamento de visibilidade e layout:
    • Remova a classe .sumir de um bloco para torná-lo visível novamente
    • Altere de inline para inline-block e veja a diferença

Exercícios: Posicionamento e Camadas com CSS

Objetivo: Praticar as propriedades position e z-index para entender como os elementos se comportam no layout.

Exercício 1 – Caixa Absoluta dentro de Caixa Relativa

  1. Crie um novo arquivo chamado exercicio-pos-1.html com um arquivo CSS separado ou estilo interno.
  2. Crie uma div com classe container (use position: relative, altura e largura definidas, cor de fundo).
  3. Dentro dela, crie outra div com classe caixa com position: absolute e posicione-a nos cantos da container.
  4. Adicione uma borda para visualizar a posição das caixas.
  5. Use top, right, bottom, left para mudar o posicionamento da caixa.

Desafio:

Crie um botão fixo no canto inferior direito da página (position: fixed) que fica visível o tempo todo com um texto como "Voltar ao topo".

Exercício 2 – Sticky Header com Z-Index

  1. Crie um novo arquivo chamado exercicio-pos-2.html.
  2. Crie um <header> no topo da página com a classe cabecalho e use position: sticky com top: 0.
  3. Dê a ele uma cor de fundo chamativa e um z-index alto para ele ficar acima do conteúdo.
  4. Crie vários parágrafos ou seções (<section>) abaixo para que seja possível rolar a página e ver o efeito "grudar".
